# 预处理器机制

CSS 预处理器的主要目标：提供 CSS 缺失的样式层复用机制、减少冗余代码，提高样式代码的可维护性。

## 基本用法

```Scss
.box {
  display: block;
}
```

## 嵌套语法

```Scss
.a {
  &.b {
    color: red
  }
}
```

## 变量

```Less
@red: #c00;

strong {
  color: @red;
}
```

```Scss
$red: #c00;

strong {
  color: $red;
}
```

## 混入

```Less
.alert {
  font-weight: 700;
}

.highlight(@color: red) {
  font-size: 1.2em;
  color: @color;
}

.heads-up {
  .alert;
  .highlight(red);
}
```

```css
.alert {
  font-weight: 700;
}
.heads-up {
  font-weight: 700;
  font-size: 1.2em;
  color: red;
}
```

```Scss
@mixin large-text {
  font: {
    family: Arial;
    size: 20px;
    weight: bold;
  }
  color: #ff0000;
}

.page-title {
  @include large-text;
  padding: 4px;
  margin-top: 10px;
}
```
