## ant design Table 控制 checkbox 不显示

```js
getCheckboxProps: (record) => {
  // style 控制是否显示可选择
  const isShow = this.filterEnableSelect(record);
  return {
    style: { display: isShow ? 'block' : 'none' },
    disabled:
      !enableInput ||
      record.isBank ||
      !isShow ||
      (record.voucherId && record.voucherCode !== '生成凭证'),
  };
},
```
