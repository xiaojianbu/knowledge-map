# 资源路径问题

## Vue-cli2

打开 config 文件夹下的 index.js，设置 assetsPublicPath: './'

## Vue-cli3

vue-cli3 在 vue.config.js 配置 publicPath，开发环境继续根目录，如下:

[publicpath](https://cli.vuejs.org/zh/config/#publicpath)

```js
module.exports = {
  publicPath:
    process.env.NODE_ENV === 'production' ? '/production-sub-path/' : '/'
}
```
